# Лабораторная работа №2  
**Тема:** Проектирование радиопокрытия мобильных сетей. Модели распространения сигналов. Бюджеты каналов.  

## Цель работы
1. Ознакомиться с методами расчёта радиопокрытия для мобильных сетей.  
2. Исследовать модели распространения сигналов: FSPL, UMINLOS, COST231, Walfish-Ikegami.  
3. Рассчитать бюджеты восходящего (UL) и нисходящего (DL) каналов.  
4. Определить уровень максимально допустимых потерь сигнала и количество базовых станций для покрытия заданной территории.  

## Ход работы

### 1. Входные данные
- Мощность базовой станции: `P_BS = 46 дБм`  
- Число секторов: `N_cells = 3`  
- Усиление антенн: `G_BS = 21 дБи`  
- Коэффициент шума: `NF_BS = 2.4 дБ`  
- Мощность UE: `P_UE = 24 дБм`, `NF_UE = 6 дБ`  
- Частота: `f = 1.8 ГГц`  
- Ширина полос: `BW_UL = 10 МГц`, `BW_DL = 20 МГц`  
- SINR: `SINR_DL = 2 дБ`, `SINR_UL = 4 дБ`  
- Потери в кабеле и дополнительные коэффициенты: `feeder_loss = 2.9 дБ`, `mimo_gain = 3 дБ`, `M_pen = 15 дБ`, `M_int = 1 дБ`  
- Площадь покрытия: `S = 96 км²`, площадь фемтосот: `S_in = 4 км²`  

---

### 2. Расчёт шумовой мощности
- Для базовой станции: `N_BS = -174 + 10*log10(BW_UL) + NF_BS`  
- Для пользовательского оборудования: `N_UE = -174 + 10*log10(BW_DL) + NF_UE`  
- Рассчитаны чувствительности приёмника: `RxSens_BS` и `RxSens_UE`.  

---

### 3. Модели распространения сигналов
- **FSPL (Free Space Path Loss)**  
- **UMINLOS (Urban Micro Non-Line-of-Sight)**  
- **COST231**  
- **Walfish-Ikegami**  

Для каждой модели вычислены потери сигнала PL на дистанциях от 10 м до 5 км.  

---

### 4. Расчёт бюджета каналов
- **Восходящий канал (UL)**: рассчитан `MAPL_UL` — максимально допустимые потери.  
- **Нисходящий канал (DL)**: рассчитан `MAPL_DL`.  
- Графически показаны горизонтальные линии `MAPL_UL` и `MAPL_DL` на фоне графиков потерь для всех моделей.  

---

### 5. Определение радиусов покрытия и количества сайтов
- Радиусы покрытия для каждой модели:  
  - `R_UMI = 441 м`  
  - `R_COST = 574 м`  
  - `R_WALFISH = 315 м`  
- Количество макросайтов и фемтосот для покрытия заданной площади:  
  - Макросайты COST: `N_site = 20` (пример)  
  - Фемтосоты UMINLOS: `N_site_in = 4` (пример)  

---

### 6. Визуализация
- Построен график зависимости потерь сигнала от расстояния для всех моделей.  
- Горизонтальные линии показывают максимально допустимые потери сигналов UL и DL.  

---
