# Лабораторная работа №3  
**Тема:** Корреляция дискретных сигналов  

## Цель работы
1. Ознакомиться с понятием корреляции и нормализованной корреляции сигналов.  
2. Научиться вычислять корреляцию между дискретными сигналами и оценивать их взаимное влияние.  
3. Исследовать корреляцию с использованием сдвигов сигналов и её визуализацию.  

## Ход работы

### 1. Вычисление корреляции для дискретных массивов (C++)
- Определены три массива: `a`, `b`, `c`.  
- Вычислена корреляция и нормализованная корреляция для всех пар массивов.  
- Построены таблицы значений корреляций.  

### 2. Сигналы во временной области (MATLAB)
- Сгенерированы три гармонических сигнала:
  - `s1_t = cos(2*pi*f1*t)`  
  - `s2_t = cos(2*pi*f2*t)`  
  - `s3_t = cos(2*pi*f3*t)`  
- Сформированы комбинации сигналов:  
  - `a_t = 2*s1_t + 4*s2_t + s3_t`  
  - `b_t = s1_t + 1/3*s2_t`  
- Вычислена корреляция и нормализованная корреляция между `s1_t` и комбинациями `a_t`, `b_t`.  

### 3. Сравнение и визуализация массивов
- Визуализированы массивы `a[]` и `b[]`.  
- Рассчитана нормализованная корреляция со сдвигами.  
- Построен график зависимости корреляции от величины сдвига.  
- Визуализирован сигнал `b[]` после сдвига на 2 элемента и сравнение с `a[]`.
