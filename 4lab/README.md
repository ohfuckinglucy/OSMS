# Лабораторная работа №4  
**Тема:** Изучение корреляционных свойств последовательностей, используемых для синхронизации в сетях мобильной связи

## Цель работы
1. Получить представление о псевдослучайных двоичных последовательностях (PN-последовательностях).
2. Изучить корреляционные свойства последовательностей Голда.
3. Реализовать генерацию PN-последовательностей и вычисление автокорреляции/взаимной корреляции в C++ и MATLAB.
4. Научиться использовать корреляционные функции для задач синхронизации в мобильных сетях.

## Ход работы

### 1. Генерация последовательности Голда (C++)
- На основе номера в журнале (x = 20 → `10100`, y = x + 7 → `11011`) инициализированы два 5-битных LFSR согласно схеме на рис. 4.4.
- Реализован цикл генерации 31-битной последовательности Голда с использованием XOR выходов регистров.
- Выведена сгенерированная последовательность.

### 2. Автокорреляция (C++)
- Реализован циклический сдвиг последовательности на все возможные значения (τ = 0…30).
- Для каждого сдвига вычислена автокорреляция по формуле (4.3):
  - `R_x(tau) = 1/p * (совпадения - несовпадения)`  
- Сформирована таблица, содержащая сдвинутые последовательности и значения автокорреляции.

### 3. Взаимная корреляция (C++)
- Сформирована вторая последовательность Голда с начальными значениями x_2 = x + 1, y_2 = y − 5 (`10101`, `10110`).
- Вычислена взаимная корреляция двух последовательностей по той же формуле (4.3).

### 4. Реализация в MATLAB
- Повторены шаги 1–4 с использованием MATLAB.
- Для вычисления корреляций использованы встроенные функции `xcorr()` и `autocorr()`, как того требует задание.
- Построен график автокорреляционной функции в зависимости от задержки (lag).

### 5. Сравнение результатов
- Проведено сравнение значений, полученных в C++ (по формуле 4.3) и в MATLAB (через `xcorr`/`autocorr`).
- Объяснены различия, связанные с разными определениями корреляции в статистике и теории сигналов.
